Welcome to Parasites Web

The main purpose of this website is to help pharmaceutical students learn about parasites.

For that task, we have developed a new AI that recognizes parasites and a game to help train that AI with your knowledge.

In order to use it, you need to register or login into your account.

Uploads

You can provide an image to the AI and it will try to identify it. If it is successfull, the AI will return you the name, otherwise, it will be added to the game so users can identify it.

Game

In this game you will be given a picture from which you can identify several parasites and name them. Your contribution will be considered among many others to improve the AI.
